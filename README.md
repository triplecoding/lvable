# HeyMariner Frontend (Next.js App Router)

## Run

```bash
npm install
npm run dev
```

## Structure

- `app/` routes and pages
- `components/` reusable UI components
- `lib/` API client, types, and fallback mock data

Set `NEXT_PUBLIC_API_BASE_URL` to your backend root (e.g. `https://api.heymariner.com`).
